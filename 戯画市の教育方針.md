# 戯画市教育委員会 教育方針（令和8年度）

※本資料は研修用の架空の自治体資料です。戯画県戯画市は実在しません。

## 1　基本理念

戯画市教育委員会は、「変化を恐れず、自ら未来を切り拓く子どもの育成」を基本理念とする。
本市は、GIGAスクール構想の全国展開当初から一人一台端末の効果的活用を市の重点施策と位置づけ、令和8年度からは生成AIの活用についても、教育委員会として積極的に推進する方針を掲げている。

教員が生成AIを「使われる」のではなく「使いこなす」存在となり、子ども一人ひとりの学びを深めるための伴走者となることを目指す。

## 2　育てたい児童・生徒像

1. **自分で考え、AIと対話しながら学びを深める子**
   生成AIを含む多様な情報源を主体的に活用し、自分の考えを広げ、深めることができる。
2. **失敗を恐れず、まず試してみる子**
   新しいツールや方法に対して臆せず挑戦し、試行錯誤の中から学びを得る。
3. **他者と協働し、多様な考えを尊重できる子**
   AIの提案を鵜呑みにせず、友達や教師との対話を通じて考えを検証し、自分の言葉でまとめることができる。
4. **情報を批判的に読み解き、責任をもって発信できる子**
   生成AIが出力する情報の正確性や著作権・個人情報への配慮を理解し、適切に判断できる。

## 3　GIGAスクール構想・生成AI活用の推進方針

- 令和8年度より、市内全教職員に生成AI（Gemini等）のアカウントを配布し、校務・授業双方での活用を推進する。
- 「戯画市生成AI活用ガイドライン」に基づき、個人情報保護・著作権に配慮した利用を徹底する。
- 各校に「AI活用推進リーダー」を1名配置し、校内研修・実践事例の共有を担う。
- 授業においては、生成AIを「答えを出す道具」ではなく「思考を深めるための対話相手」として位置づける。

## 4　戯画市授業スタンダード

戯画市が全校共通で目指す授業の姿を、以下の4点に整理する。

| 観点 | スタンダードの内容 |
| :---- | :---- |
| ① めあて・見通し | 児童生徒が「何のために学ぶか」を自分の言葉で説明できる導入を行う |
| ② 対話・協働 | 一人で考える時間と、他者（人・AI）と対話する時間を意図的に組み合わせる |
| ③ 個別最適な学び | 生成AIやデジタル教材を活用し、一人ひとりの進度・興味に応じた学びを保障する |
| ④ 振り返り | 学びの過程を自分の言葉やポートフォリオで振り返り、次の学びにつなげる |

## 5　数値目標（令和8年度末）

- 生成AIを校務に「週1回以上」活用する教員の割合：80％以上
- 授業で生成AIを活用した実践事例の共有数：各校年間5件以上
- 「学びに前向きだ」と回答する児童生徒の割合：90％以上（学校評価アンケート）
