# 穴炉愚市教育委員会 教育方針（令和8年度）

※本資料は研修用の架空の自治体資料です。穴炉愚県穴炉愚市は実在しません。

## 1　基本理念

穴炉愚市教育委員会は、「手と心を使い、確かな学力と人間性を育む教育」を基本理念とする。
本市は、デジタル機器に頼らず、実物に触れ、自分の手で書き、体を動かす経験こそが子どもの根っこを育てるという考えのもと、市内小中学校において端末利用を最小限にとどめる方針を継続している。

生成AIをはじめとするデジタルツールについては、思考力・表現力の土台が育つ前に安易に頼ることを避け、まずは紙と鉛筆、対話、体験を通じた学びを徹底することを重視する。

## 2　育てたい児童・生徒像

1. **自分の手で書き、考えをまとめられる子**
   ノートに自分の言葉で考えを書き出し、文字や図で整理する力を大切にする。
2. **粘り強く、最後まで自分で考え抜く子**
   すぐに答えを求めず、試行錯誤しながらじっくり考え抜く姿勢を育てる。
3. **人と向き合い、直接対話できる子**
   画面越しではなく、相手の表情や声を通じて気持ちを汲み取り、対話する力を育てる。
4. **自然や実物に触れ、五感で学ぶ子**
   本物の教材・自然物・体験活動を通じて、実感を伴う学びを積み重ねる。

## 3　デジタル機器・生成AIの利用方針

- 小中学校における端末利用は、必要最小限（年数回の調べ学習等）にとどめる。
- 生成AIの児童生徒による利用は、原則として認めない。
- 教職員についても、校務における生成AI利用は各校長の判断のもと慎重に検討することとし、市として積極的な推進は行わない。
- 家庭に対しても、低学年のうちはデジタル機器よりも紙の本・対話・外遊びの時間を大切にするよう呼びかける。

## 4　穴炉愚市授業スタンダード

穴炉愚市が全校共通で目指す授業の姿を、以下の4点に整理する。

| 観点 | スタンダードの内容 |
| :---- | :---- |
| ① めあて・板書 | 教師が黒板に丁寧に板書し、児童生徒は板書を必ずノートに完全書き写しする（要約・省略は認めない） |
| ② 対話・音読 | 机を向き合わせ、顔を見て話す・聞く・音読する活動を毎時間必ず組み込む |
| ③ 反復・鍛錬 | 計算・漢字・作文などの基礎基本を、繰り返し書く・唱えることで定着させる |
| ④ 振り返り | 学習の振り返りは紙の「振り返りノート」に手書きで記録し、教師が朱書きで応答する |

## 5　数値目標（令和8年度末）

- 家庭学習における「紙のノート・ドリル」使用率：90％以上
- 授業中の対話・音読活動を「毎時間行っている」学級の割合：100％
- 「文字を丁寧に書けている」と回答する保護者の割合：85％以上（学校評価アンケート）
